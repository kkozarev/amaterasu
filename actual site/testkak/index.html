Spiral segment chart

Move your mouse over it for interactivity

A spiral segment chart displays quantitative data as an array of spiral segments, coloured according to value.

This is implemented using D3.js and is reusable.

The code to create the above chart is:

var chart = spiralSegmentChart()
    .innerRadius(20)
    .yearLabels(["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"])
    .monthLabels(["January","February","March","April","May","June","July","August","September",
                    "October","November","December"]);

var spiralData = [ 0.176042, 0.096146, 0.076414, ..more data.. , 0.192478, 0.134997, 0.152271 ];

d3.select('#spiralchart')
    .selectAll('svg')
    .data([spiralData])
    .enter()
    .append('svg')
    .call(chart);
